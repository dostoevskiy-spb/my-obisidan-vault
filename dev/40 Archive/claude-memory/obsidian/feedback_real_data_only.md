При тестировании skills и функций vault — использовать РЕАЛЬНЫЕ данные, не тестовые заглушки.

**Why:** Vault — это Second Brain, туда попадает только настоящая информация. Тестовые записи засоряют базу знаний.

**How to apply:** При тестировании vault-task, capture и т.д. — использовать текущую реальную задачу/заметку. Не "протестировать X", а реальное описание того чем занимаемся.
