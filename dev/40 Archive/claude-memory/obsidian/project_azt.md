Помогаем другу с сайтом azt-teplica.ru (Александровский Завод Теплиц, Пермь).

**Why:** Дружеская помощь, не коммерческий проект. **Основной поисковик для продвижения — Яндекс**, не Google.

**How to apply:** Все материалы проекта в `10 Projects/AZT/`. Полный аудит проведён 2026-04-12, выявлены критические проблемы: сломанные title на региональных страницах (3 из 4 городов показывают "Пермь"), отсутствие meta description на 90% страниц, дубли без canonical, шаблонные юрстраницы (риск Минусинска), системный копипаст тепличного контента на всех категориях (риск Баден-Бадена), минимальная Schema.org. Приоритеты под Яндекс: Вебмастер + региональная привязка, Яндекс.Бизнес для каждого города, отзывы на Картах, Schema LocalBusiness/Review/FAQ, скорость загрузки. Сайт на Битриксе, разработчик Maximus Media.

**Текущее состояние (2026-04-15):** Аудит завершён, документация готова, проект в стадии исполнения. Технические доработки (разработчик) и подготовка статей (SEO-агентство) идут параллельно. Техничка займёт ~2 недели, после её завершения готовые статьи сразу выкладываются. Медиаплан и контент-план уже есть, статьи пишутся. Платный трафик (Яндекс Директ) работает с небольшим бюджетом, не отключается. Павел получит MCP-доступ к Вебмастеру, Метрике и Вебвизору.
