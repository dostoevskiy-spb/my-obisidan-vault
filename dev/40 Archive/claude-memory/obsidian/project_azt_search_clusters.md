**Why:** агентство утверждало про «работающие группы капелька → арочные после скидки/акции»; данные подтверждают только капельку, арочные за 90 дней дали ноль конверсий и ноль показов >20.

**How to apply:**
- Если агентство снова ссылается на «арочные как сильную группу» — спрашивать ИХ отчёт по AdGroupName + Query вложенно (возможно, в их группе с названием «Арочная» падают общие запросы по широкому матчу, а не реальные «арочн»).
- Капелька — точечный длинный хвост (запросы с 2–9 показами), не массовая категория. Пуш в отдельную кампанию с приоритетом разумен (CR 87%), но потолок объёма низкий.
- Основные деньги делает «прочее» (81%): «теплицы пермь», «купить теплицу в тюмени» и т.п. — гео+общие+поликарбонат.
- Скидка/акция работает, но в 1,6× дороже капельки и в 2× хуже по CR — не «звезда», как преподносят.
