# Яндекс Директ — где теряются деньги, а где нет

## Что автоматически НЕ стоит денег (антифрод возвращает)

Яндекс Директ автоматически фильтрует и возвращает на баланс оплату за:

- **Фродовый трафик** — клики роботов, скликивание конкурентами, организованные схемы накрутки
- **Случайные клики** — mistap'ы, технические ошибки
- **Недействительные показы и конверсии** (роботные)

Источник: https://yandex.ru/support/direct/ru/technologies-and-services/antifraud

Возвраты автоматические, не требуют действий. Видны в Мастере отчётов → «Недействительные клики и конверсии». Проверка назад — только 30 дней.

## Что ТОЧНО списывается и НЕ возвращается

Антифрод сознательно НЕ фильтрует (это не ошибка, это бизнес-правило):

- **Высокий bounce rate / отказы** — быстрый уход реального человека считается легитимным кликом
- **Реальные люди на нерелевантных объявлениях** (плохой matching) — не фрод
- **Клики из мусорных РСЯ-площадок** — если Яндекс не считает их роботными, деньги списаны
- **Опечатки и криво заполненные формы** — ошибка пользователя, не фрод
- **Повторные конверсии** одного человека с разных визитов — валидны

**Вывод:** высокий процент отказов = прямая финансовая потеря без возврата. Для рекламодателя это проблема настройки кампании, не Яндекса.

## Три модели оплаты и их риски

### 1. CPC (оплата за клик) — по умолчанию
Платишь за каждый клик. При 40% отказов → 40% бюджета уходит впустую. Самая рискованная модель.

### 2. CPA (оплата за конверсию) — только через автостратегию «Максимум конверсий»
- Платишь только за целевое действие (форма, звонок)
- **Нет конверсий = нет списаний**
- Требования:
  - Минимум 10 конверсий в неделю
  - Цена конверсии ≤ 5 000 руб
  - Недельный бюджет = 70× цена (минимум) или 350× цена (рекомендация)
  - На балансе — 5 000 руб или 3× цена конверсии
- Атрибуция 21 день

Источник: https://yandex.ru/adv/news/model-oplaty-za-konversii-v-yandeks-direkte-dostupna-vsem-reklamodatelyam

### 3. Максимум кликов / Оптимизация кликов
Автостратегия гребёт дешёвые клики. В нашем кейсе (AZT, 18.04.2026) именно такая стратегия Мастера Кампаний разогнала РСЯ вечером и слила `~15-20 тыс руб` за 4 часа.

## Возврат средств — не за плохой трафик, а со счёта

https://yandex.com/support/direct/ru/payments/refund.html

Это механизм вывода остатка, не компенсация. Условия:
- Остановить кампанию
- Подать заявку с номером счёта
- 20+ рабочих дней обработки
- Возвращается только реально внесённая сумма (без бонусов), на исходный способ оплаты
- Для юрлица — письмо за подписью директора и печатью

Оспорить списание за низкокачественный трафик РСЯ **нельзя** — только через антифрод-фильтр, который работает автоматически.

## Мастер Кампаний — финансовые особенности

https://direct.yandex.ru/ads/products/master-kampanij

- Отдельный раздел кабинета, не в общей вкладке «Кампании»
- Показывается везде: Поиск + РСЯ + Карты + Бизнес — **алгоритм сам выбирает**
- Поддерживает оплату за конверсии («Максимум целевых действий по средней цене за неделю»)
- Без ограничения площадок → легко скатывается в мусорные РСЯ
- В кабинете Директа его расходы видны отдельно — при проверке «сколько потратили» обычной вкладки недостаточно

**Безопасный запуск Мастера Кампаний:**
- Стратегия «Оплата за конверсии»
- Цена конверсии ниже реальной (Яндекс не сможет разогнаться на мусоре)
- Минимум 10 конверсий в неделю — иначе алгоритм не обучится

## Отключение площадок РСЯ — основной инструмент экономии

https://yard.yandex.ru/courses/direct-prodvinutyy/otkluchenie-reklamnyh-ploschadok

- Лимит: 1 000 площадок в Performance-кампании, 40 в графической
- Нельзя отключить ya.ru и региональные домены Яндекса
- Минус-слова в РСЯ не работают — только минус-площадки
- Алгоритм «контроля качества» Яндекса снижает ставки на мусорных площадках автоматически, но не отключает их полностью

**Как чистить:**
1. Отчёт Мастера отчётов → dimension = площадка, metrics = клики/CTR/конверсии/отказы
2. Искать аномалии: траты без конверсий, CTR сильно ниже среднего, тематически нерелевантные названия
3. Массово отключать мобильные приложения РСЯ через настройки (отдельный переключатель)

## Расчётная оценка потерь на плохой настройке

При типовом CPC 30-50 руб и 40% отказов в Перми/Тюмени:
- 100 визитов с отказами 40% = `40` впустую × `40 руб` = **`1 600 руб/100 визитов` слив**
- 500 визитов в день (как AZT 18.04) = **`~8 000 руб/сутки` слив** только от отказов
- В пиковую неделю апреля = **до `56 000 руб/неделю`** чистой утечки

Это минимум. Реальная потеря больше — ещё упущенные заявки, которые могли быть при правильной посадочной.

## Быстрые проверки на финансовые потери

1. **Метрика → Отчёт «Директ, сводка»**: сравнить визиты и конверсии по каждой кампании. Аномально высокие расходы без конверсий = красный флаг.
2. **Директ → Мастер отчётов → Площадки**: отсортировать по расходу desc, отказам desc. Топ-10 с отказами >50% — кандидаты на отключение.
3. **Директ → Недействительные клики**: проверить, возвращает ли Яндекс хоть что-то. Обычно 3-7% трафика уходит в фрод.
4. **Директ → вкладка «Мастер кампаний»**: если там есть активные кампании — проверить стратегию. Если не «Оплата за конверсии» — это потенциальная утечка.
