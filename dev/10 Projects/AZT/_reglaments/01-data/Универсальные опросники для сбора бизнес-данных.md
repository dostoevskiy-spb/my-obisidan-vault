# Универсальные опросники для сбора бизнес-данных

## Назначение

Опросники нужны, чтобы получить от бизнеса или команды бизнес-контекст, без которого нельзя точно оценить коммерческую эффективность SEO, Директа, сайта, карточек и контента.

Цель опросника — понять не просто "как больше привести трафика", а какие обращения, регионы, товары и каналы действительно выгодны бизнесу.

## Категории опросников

Для разных типов бизнеса нужны разные версии.

Черновой набор категорий:

- `Розница / товары / e-commerce / локальные продажи товаров`;
- `Услуги и лидогенерация`;
- `B2B / производство / оптовые продажи`;
- `Локальный бизнес с точками / филиалами`;
- `Маркетплейсы и мультиканальная торговля`;
- `Сезонный бизнес`;
- `Сложные услуги с длинным циклом сделки`;
- `Инфобизнес / обучение / мероприятия`.

Ниже подробно заполнен универсальный опросник для розницы и товарного бизнеса. Он подходит как основа для теплиц, шин, мягких игрушек, мебели, стройматериалов, оборудования и других товарных направлений.

## Универсальный опросник: розница / товары

### 1. Бизнес и цели

1. Что продаете: основные категории товаров?
2. Какие категории самые приоритетные для роста?
3. Какие категории самые маржинальные?
4. Какие категории дают много заявок, но мало прибыли?
5. Какие категории сейчас не хотите продвигать?
6. Какие товары сезонные?
7. Какие товары всегда доступны?
8. Есть ли товары, которые часто заканчиваются?
9. Что для вас важнее сейчас: больше заявок, больше продаж, выше средний чек, меньше стоимость лида, рост в регионах, узнаваемость?
10. Какой результат через 1 месяц, 3 месяца и 6 месяцев будет считаться успешным?

### 2. География

1. В каких регионах продаете сейчас?
2. Какие регионы самые приоритетные?
3. Какие регионы самые прибыльные?
4. В какие регионы невыгодно продавать?
5. Где есть доставка?
6. Где есть самовывоз, шоурум, склад или точка продаж?
7. Где есть монтаж, установка или сервис?
8. Есть ли регионы, где спрос есть, но бизнес пока не готов его обрабатывать?
9. Есть ли разные условия доставки/монтажа по регионам?
10. Какие города обязательно должны быть в рекламе и SEO?

### 3. Клиенты и качество лидов

1. Кто основной покупатель: частные лица, компании, дилеры, строители, закупщики?
2. Какие клиенты самые желательные?
3. Какие клиенты нежелательные?
4. Какие вопросы чаще всего задают перед покупкой?
5. Какие возражения чаще всего мешают покупке?
6. Что обычно важно клиенту: цена, доставка, гарантия, отзывы, наличие, бренд, установка, рассрочка?
7. Какие заявки считаются целевыми?
8. Какие заявки считаются нецелевыми?
9. Какие признаки хорошего лида?
10. Какие признаки плохого лида?

### 4. Продажи и экономика

1. Есть ли данные по среднему чеку?
2. Есть ли данные по маржинальности категорий?
3. Есть ли минимальная сумма заказа, при которой продажа выгодна?
4. Есть ли товары-локомотивы, которые привлекают клиента, но не дают основную прибыль?
5. Есть ли допродажи: доставка, монтаж, аксессуары, гарантия, комплектующие?
6. Какой процент заявок превращается в продажу?
7. Какой процент звонков превращается в продажу?
8. Какая стоимость лида считается приемлемой?
9. Какая стоимость продажи считается приемлемой?
10. Есть ли сезонные изменения среднего чека и маржинальности?

### 5. Каналы привлечения

1. Какие каналы сейчас дают заявки: Директ, SEO, Яндекс Бизнес, Авито, соцсети, рекомендации, офлайн?
2. Какие каналы дают самые качественные заявки?
3. Какие каналы дают много мусора?
4. Какие каналы дорогие, но нужные?
5. Какие каналы хотели бы развивать?
6. Есть ли понимание текущей стоимости лида по каналам?
7. Есть ли понимание текущей стоимости продажи по каналам?
8. Есть ли каналы, которые нельзя отключать?
9. Есть ли каналы, которые хотите проверить как гипотезу?
10. Какие каналы уже пробовали и отказались?

### 6. Реклама

1. Какие товары сейчас рекламируются?
2. Какие регионы рекламируются?
3. Какие посадочные используются?
4. Какие объявления или офферы лучше работают?
5. Какие запросы точно не нужны?
6. Какие минус-слова очевидны бизнесу?
7. Какие акции, скидки, рассрочки можно использовать в объявлениях?
8. Есть ли ограничения по обещаниям в рекламе?
9. Какие рекламные лиды считаются качественными?
10. Как быстро менеджеры обрабатывают заявки из рекламы?

### 7. SEO и контент

1. Какие темы клиенты ищут перед покупкой?
2. Какие вопросы менеджеры чаще всего объясняют вручную?
3. Какие статьи или страницы уже помогают продажам?
4. Какие статьи точно не нужны?
5. Какие товары требуют объяснения выбора?
6. Какие сравнения важны: модели, материалы, размеры, бренды, комплектации?
7. Какие темы могут привести нецелевой информационный трафик?
8. Какие темы должны вести к коммерческой посадочной?
9. Какие товары или категории требуют отдельной страницы?
10. Какие страницы нужно обновить перед написанием новых статей?

### 8. Сайт и конверсия

1. Какие страницы сейчас самые важные?
2. Какие страницы должны давать заявки?
3. Где пользователи чаще всего задают вопросы или путаются?
4. Какие формы заявок используются?
5. Какие способы связи важны: звонок, форма, WhatsApp, Telegram, VK, email?
6. Есть ли проблемы с обработкой заявок с сайта?
7. Есть ли страницы, которые бизнес считает слабыми?
8. Какие элементы доверия важны: отзывы, фото, сертификаты, гарантия, производство, адрес?
9. Какие возражения сайт должен закрывать до обращения?
10. Что пользователь должен сделать после чтения статьи?

### 9. Операционные ограничения

1. Есть ли ограничения по складу?
2. Есть ли ограничения по доставке?
3. Есть ли ограничения по монтажу?
4. Есть ли ограничения по персоналу/обработке заявок?
5. Есть ли периоды, когда нельзя масштабировать рекламу?
6. Есть ли товары, которые нельзя быстро поставить?
7. Есть ли регионы, куда нельзя быстро доставить?
8. Есть ли юридические или рекламные ограничения?
9. Есть ли сезонные пики, к которым нужно готовиться заранее?
10. Какие задачи бизнеса сейчас важнее маркетинга?

### 10. Данные, которые можно предоставить

1. Есть ли выгрузка заявок?
2. Есть ли выгрузка звонков?
3. Есть ли CRM?
4. Есть ли статусы лидов?
5. Есть ли продажи по источникам?
6. Есть ли средний чек по категориям?
7. Есть ли маржинальность по категориям?
8. Есть ли список приоритетных товаров?
9. Есть ли список нежелательных товаров/регионов/клиентов?
10. Есть ли данные по сезонности прошлых лет?

## Черновик: услуги и лидогенерация

Дополнительно к универсальным вопросам нужно выяснить:

- тип услуги;
- этапы сделки;
- средняя длительность сделки;
- что считается квалифицированным лидом;
- какие услуги самые маржинальные;
- какие услуги требуют консультации;
- какие услуги можно продать сразу;
- какие услуги требуют выезда, замера, диагностики;
- какие специалисты ограничивают масштабирование;
- какие заявки не берутся в работу.

## Черновик: B2B / производство / опт

Дополнительно выяснить:

- минимальный заказ;
- тип покупателя;
- цикл сделки;
- регионы поставки;
- дилеры и партнеры;
- оптовые цены;
- производственные ограничения;
- сроки изготовления;
- типовые возражения закупщиков;
- документы, сертификаты, тендеры.

## Черновик: локальный бизнес с точками

Дополнительно выяснить:

- список точек;
- зоны обслуживания;
- карточки Яндекс Бизнеса;
- рейтинг и отзывы;
- локальные конкуренты;
- маршруты;
- звонки из карточек;
- роль сайта для каждой точки;
- различия по ассортименту и условиям;
- кто отвечает за карточки.
