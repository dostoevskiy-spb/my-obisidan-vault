# Анализ Wordstat и спроса

## Назначение

Wordstat используется для оценки потенциального спроса, который сайт может получать через SEO, GEO, Яндекс Бизнес и контекстную рекламу.

Wordstat отвечает на вопрос:

```text
Что люди ищут в принципе, даже если сайт проект сейчас этот спрос не получает?
```

## Место в модели

Wordstat стоит между фактическими данными и стратегией роста.

Связка:

```text
Wordstat показывает спрос
→ Вебмастер показывает, видит ли Яндекс страницы сайта
→ Метрика показывает, получает ли сайт трафик
→ цели показывают, дает ли трафик результат
→ Директ показывает, покупаем ли мы этот спрос за деньги
```

## Кластеры спроса

Семантика делится на кластеры:

- теплицы;
- поликарбонат;
- грядки;
- доставка;
- монтаж;
- цена;
- рассрочка;
- отзывы;
- официальный сайт;
- гарантия;
- сравнение моделей;
- усиленные теплицы;
- снеговая нагрузка;
- региональные запросы;
- брендовые запросы;
- конкурентные запросы;
- информационные запросы.

## Региональность

Для каждого ключевого кластера проверяются регионы:

- Пермь;
- Тюмень;
- Екатеринбург;
- Ижевск;
- другие регионы, если они есть в фактическом спросе или планах агентства.

Региональный спрос сопоставляется с:

- региональной посадочной сайта;
- статусом страницы в Вебмастере;
- карточкой Яндекс Бизнеса;
- фактическим трафиком Метрики;
- платной рекламой;
- планами агентства.

## Статусы спроса

Каждый кластер получает статус:

- `есть спрос, сайт его получает`;
- `есть спрос, сайт его не получает`;
- `есть спрос, но страница проблемная`;
- `есть спрос, но нет посадочной`;
- `есть спрос, закрывается карточкой Яндекс Бизнеса`;
- `есть спрос, покупается рекламой`;
- `есть спрос, но нет доказанного коммерческого результата`;
- `спрос не подтвержден`.

## Сопоставление с Вебмастером

Если Wordstat показывает спрос, а Вебмастер показывает проблемы со страницей, приоритет задачи повышается.

Пример:

```text
Есть региональный спрос по теплицам в Перми.
Вебмастер фиксирует LOW_QUALITY на /prm/.
Метрика Яндекс Бизнеса показывает действия по пермской карточке.
Вывод: региональную страницу нужно проверять и исправлять до масштабирования SEO/рекламы.
```

## Сопоставление с Директом

Если спрос покупается рекламой, нужно проверить:

- есть ли органический потенциал;
- есть ли посадочная;
- нет ли технических проблем посадочной;
- есть ли конверсии;
- можно ли часть спроса со временем закрывать SEO или карточками;
- какие запросы надо оставить в платном канале.

## Выходные артефакты

Результат анализа Wordstat:

- карта спроса;
- кластеры семантики;
- региональная матрица спроса;
- список недостающих посадочных;
- список SEO/GEO-возможностей;
- список рекламных гипотез;
- вопросы к агентству по семантике и медиаплану;
- входные данные для контент-плана и ТЗ на статьи.

## Запреты

Нельзя делать вывод:

```text
В Wordstat много запросов, значит нужно писать статьи.
```

Нужно проверить:

- коммерческий или информационный это спрос;
- есть ли посадочная;
- есть ли индексирование;
- есть ли фактический трафик;
- есть ли конверсии;
- есть ли конкуренты в выдаче;
- можно ли этот спрос закрыть страницей, карточкой, статьей или рекламой.
