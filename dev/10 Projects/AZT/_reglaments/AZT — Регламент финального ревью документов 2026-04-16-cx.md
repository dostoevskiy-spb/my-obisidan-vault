# AZT — Регламент финального ревью документов

## Назначение

Финальное ревью нужно, чтобы документ выглядел профессионально, не содержал неподтвержденных выводов, лишней воды, AI-slop, эмоциональных оценок и противоречий.

## Чеклист доказательности

Перед отправкой проверить:

- каждый сильный вывод имеет источник;
- у каждой цифры есть период;
- у каждого технического тезиса есть пример или источник;
- гипотезы отделены от доказанных фактов;
- вопросы отделены от выводов;
- приоритеты соответствуют матрице доказательств;
- новые данные не противоречат старым без объяснения.

## Чеклист языка

Удалить или заменить:

- общие фразы без данных;
- обвинительный тон;
- эмоциональные слова;
- SEO-жаргон без пояснений;
- повторяющиеся формулировки;
- длинные абзацы без вывода;
- обещания эффекта без механики.

Примеры замены:

```text
Было: Это негативно влияет на SEO.
Стало: Страница исключена из поиска, поэтому не может получать органический трафик из Яндекса.
```

```text
Было: Нужно срочно исправить.
Стало: Задача имеет P0, потому что Вебмастер фиксирует исключение URL, а Метрика показывает спрос в связанном регионе.
```

## Чеклист адресата

Для собственника:

- коротко;
- понятен бизнес-смысл;
- нет перегруза API-деталями;
- ясно, что делать дальше;
- есть риски бездействия.

Для разработчика:

- конкретные URL;
- конкретное ожидаемое поведение;
- критерии приемки;
- способ проверки.

Для агентства:

- факты;
- вопросы;
- ожидание KPI;
- без эмоциональных оценок.

## Проверка противоречий

Перед финализацией нужно сверить:

- отчет собственнику;
- анализ Вебмастера;
- данные Метрики;
- ТЗ разработчику;
- план развития;
- вопросы агентству.

Если один документ говорит "сначала техничка", а другой предлагает запуск контента без оговорок, это противоречие нужно устранить.

## Правило последнего прохода

Последний проход проводится как code review:

- сначала искать ошибки и риски;
- потом проверять структуру;
- потом чистить язык;
- потом проверять ссылки и цифры.

Нельзя начинать финальное ревью с косметики, пока не проверена доказательная база.
