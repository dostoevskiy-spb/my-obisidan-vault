# Словарь метрик и сущностей

## Назначение

Документ фиксирует единый язык проекта.

Он нужен, чтобы бизнес, аналитик, агентство и разработчик одинаково понимали ключевые термины: лид, конверсия, посадочная, P0, baseline, доказанный факт, гипотеза, влияние.

## Статусы доказанности

| Термин | Значение |
|---|---|
| Факт | проверенное наблюдение с источником и периодом |
| Гипотеза | предположение, которое требует проверки |
| Подтверждено | вывод проверен по заданному критерию |
| Не подтвердилось | ожидаемая связь или эффект не найден |
| Недостаточно данных | данных не хватает для вывода |
| Невозможно отделить влияние | есть слишком много параллельных факторов |

## Воронка

| Термин | Рабочее определение |
|---|---|
| Показ | пользователь увидел сайт, карточку или объявление в источнике |
| Клик | переход из источника на сайт или в карточку |
| Визит | сессия пользователя на сайте |
| Посадочная страница | первая или целевая страница, на которую приходит пользователь |
| Микроконверсия | действие, показывающее интерес, но не равное полноценному лиду |
| Коммерческое действие | звонок, форма, маршрут, переход в мессенджер или другое действие с коммерческим намерением |
| Лид | обращение, которое можно обработать как потенциальную продажу |
| Качественный лид | лид, соответствующий критериям бизнеса |
| Продажа | подтвержденная сделка или оплата |

## Каналы

| Термин | Рабочее определение |
|---|---|
| SEO | органический поиск по сайту |
| GEO | региональная видимость сайта и посадочных |
| Яндекс Бизнес | карточки организации в сервисах Яндекса |
| Директ | платный трафик из Яндекс Директа |
| SERP | поисковая выдача |
| Wordstat | источник оценки спроса и сезонности |
| Вебвизор | инструмент анализа поведения пользователя по сессиям |

## Технические сущности

| Термин | Рабочее определение |
|---|---|
| Sitemap | файл или набор файлов, через которые сайт сообщает поисковику URL для обхода |
| Canonical | указание основной версии страницы |
| 404 | страница не найдена |
| 5xx | серверная ошибка |
| `LOW_QUALITY` | сигнал Яндекса о низком качестве или недостаточной ценности страницы |
| `NOT_CANONICAL` | URL исключен как неканонический |
| Baseline | зафиксированное состояние до изменения |
| Контрольное окно | период после изменения, за который проверяется эффект |

## Приоритеты

| Термин | Рабочее определение |
|---|---|
| P0 | блокер или срочная проблема, мешающая индексации, измерению, рекламе или коммерческим действиям |
| P1 | важная задача, влияющая на рост, качество трафика, конверсии или управляемость |
| P2 | улучшение, которое полезно, но не блокирует текущий цикл |

## Деньги и эффективность

| Термин | Рабочее определение |
|---|---|
| CPL | стоимость лида |
| CPA | стоимость целевого действия или продажи, если продажа измеряется |
| CPC | стоимость клика |
| CTR | кликабельность объявления или элемента |
| ROMI | возврат маркетинговых инвестиций, если есть данные по выручке/марже |
| Стоимость ошибки | деньги или возможности, потерянные из-за неверного действия или технической проблемы |

## Правило

Если термин используется в отчете, ТЗ или вопросах агентству в значении, отличающемся от этого словаря, значение нужно явно уточнить в документе.
