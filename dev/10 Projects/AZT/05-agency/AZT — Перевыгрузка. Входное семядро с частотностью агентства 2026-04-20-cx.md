# AZT — Входное семядро с частотностью агентства

## Назначение

Документ фиксирует получение перевыгрузки семантического ядра с частотностью и разбивкой по городам (Пермь, Тюмень), подготовленной агентством.

Это не оценка качества ядра и не согласование. Это реестр входного файла, который дальше нужно проверить: корректны ли частотности, полон ли охват, релевантны ли минус-фразы, как соотносится с предыдущим ядром [[AZT — Входное семантическое ядро агентства 2026-04-17-cx]].

Важный контекст: к каждому городу собрана основная семантика + «расширенная из поисковых запросов, сегодня». Минус-фразы собраны утром 20 апреля из поисковых запросов за два дня работы кампаний.

## Что внутри

**Пермь и Тюмень** — по 4 колонки:
1. Основной запрос (из стартового ядра)
2. Частотность основного
3. Расширенный запрос (собран из фактических поисковых запросов, 20 апреля)
4. Частотность расширенного

**Минус-фразы** — 10 слов, собранных из поисковых запросов за два дня работы кампаний: атмосферная, беседка, бытовка, гудвин, забор, ладно, максидом, металлострой, нужно, складно.

**Инфо** содержит комментарии агентства:
- поисковые кампании + планируемое расширение семантики по факту;
- кампании в РСЯ без автотаргетинга, далее — аудитории и чистка площадок;
- Мастера кампаний с оплатой за конверсии («Страница спасибо»);
- в качестве посадочных — страницы каталога теплиц, в планах — МК с посадочной «Главная».

## Как использовать дальше

- сравнить семядро с предыдущим [[AZT — Входное семантическое ядро агентства 2026-04-17-cx]] — что добавилось, что убралось;
- сопоставить расширенную семантику с «Рабочей таблицей по контекстной рекламе» [[AZT — Входная рабочая таблица по контекстной рекламе агентства 2026-04-08-cx]] и «Медиапланом» [[AZT — Входной медиаплан агентства 2026-04-08-cx]];
- оценить адекватность 10 минус-фраз и пополнить свой список по итогам анализа поисковых запросов;
- свериться с Wordstat по частотностям для Перми и Тюмени;
- сформировать вопросы агентству: почему нет Екатеринбурга, Ижевска, Москвы (которые есть в медиаплане), но есть только Пермь+Тюмень в этом ядре;
- перенести подтверждённые наблюдения в матрицу доказательств.
