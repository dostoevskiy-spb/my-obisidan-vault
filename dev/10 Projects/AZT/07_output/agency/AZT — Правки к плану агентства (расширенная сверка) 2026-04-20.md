## 3. Минус-фразы — требуется пересмотр по официальной документации Яндекса

### 3.1. Как работают минус-фразы (цитата Справки Директа)

> Минус-слова и минус-фразы — это слова и словосочетания, по запросам с которыми объявление показываться не будет.
>
> Если вы добавите минус-слово «завод» без операторов, объявление не будет показано по запросу «александровский завод теплиц», так как запрос полностью содержит это слово.

Операторы:
- `-завод` (без оператора) — блокирует все словоформы (завод, заводы, заводской) и все фразы, содержащие слово в любой форме.
- `-!завод` — блокирует только точную словоформу «завод».
- `-[завод теплиц]` — блокирует точную последовательность слов.

Источники: [Яндекс Директ — Минус-слова](https://yandex.ru/support/direct/ru/keywords/negative-keywords), [Яндекс Директ — Символы и операторы](https://yandex.ru/support/direct/ru/keywords/symbols-and-operators).

- `-ремонт` без оператора отсекает «замена теплицы», но впускает мусор типа «ремонт стиральных машин», «ремонт крыши» — поэтому взамен общего минуса стоит добавить фразовые/точные: `-[ремонт стиральных]`, `-[ремонт квартиры]`, `-!ремонт` (только точная словоформа), `-[ремонт телефона]`.
- `-мини` — при снятии возможен трафик по «мини юбка», «мини отель». Подставить `-[мини юбка]`, `-[мини отель]`, `-[мини бар]`.
- `-отзывы` — при снятии нужно следить, чтобы реклама не крутилась по «отзывы сотрудников», «плохие отзывы» (отрицательные сентименты); добавить `-!плохие -!негативные` как защиту.
- `-новатор`, `-воля` (если есть) — это бренды конкурентов, снимать нельзя, только точечно дополнять минусами из п. 3.3.

**Что сделать:**
1. Пересмотреть список общих минусов, убрав `завод`, `отзывы`, `ремонт`, `демонтаж`, `домиком`, `мини`, `отапливаемая` (либо переоформить их с операторами). По `прямостенная` — сначала проверить в интерфейсе, не удалять вслепую.
2. Для каждого снятого общего минуса добавить фразовые/точные минусы против мусорного трафика (см. оговорку выше).
3. Проверить в интерфейсе Директа, что минусы распределены **по кампаниям**, а не одним общим списком для всех. Иначе минус `екатеринбург` отсекает трафик в ЕКБ-кампании и т. д.
