# Настройка OpenClaw для Second Brain

## Деплой конфигов

Черновики конфигов хранятся в этой папке (`10 Projects/Бадян/`).
Рабочие версии деплоятся в `~/.openclaw/workspace/`.

### Порядок деплоя

```bash
# Скопировать конфиги (убрать -draft из имён и markdown-обёртку)
# Каждый *-draft.md содержит блок ``` с содержимым для реального файла

# 1. Основные файлы
cp содержимое_SOUL-draft.md → ~/.openclaw/workspace/SOUL.md
cp содержимое_IDENTITY-draft.md → ~/.openclaw/workspace/IDENTITY.md
cp содержимое_USER-draft.md → ~/.openclaw/workspace/USER.md
cp содержимое_AGENTS-draft.md → ~/.openclaw/workspace/AGENTS.md
cp содержимое_TOOLS-draft.md → ~/.openclaw/workspace/TOOLS.md

# 2. Автоматизация
cp содержимое_HEARTBEAT-draft.md → ~/.openclaw/workspace/HEARTBEAT.md

# 3. Первый запуск
cp содержимое_BOOTSTRAP-draft.md → ~/.openclaw/workspace/BOOTSTRAP.md

# 4. Установить obsidian skill
openclaw install skill obsidian
```

### Что после деплоя

1. Запустить OpenClaw — он прочитает BOOTSTRAP.md
2. Пройти чеклист первого запуска
3. BOOTSTRAP.md будет удалён автоматически
4. Проверить что HEARTBEAT работает (подождать 30 мин)

## Обновление конфигов

1. Редактируй черновик в vault (Obsidian)
2. Деплой в workspace
3. Перезапусти сессию OpenClaw
