## Реализация

### Сбор данных
- Yandex Direct MCP — `direct_report(SEARCH_QUERY_PERFORMANCE_REPORT)` со всех 8 кампаний, период `2026-03-24 — 2026-04-23`, поля Query/CampaignId/CampaignName/Impressions/Clicks/Cost/Conversions
- Yandex Webmaster MCP — `ywm_query_analytics(QUERY, 500 × 3 страницы)` для хоста `https:azt-teplica.ru:443` (popular_queries падает с missing order_by, используем query_analytics)
- Yandex Metrika MCP — `get-search-phrases(counter_id=87177110, limit=100)` для главного счётчика сайта

### Кластеризация
- AWK-скрипт на `/tmp/cluster_apply.sh` — кластеры по тексту запроса: brand (азт/александровск), glass (стекл), drop (капля/капел), vertical (прямостен), arch (арочн), breadbox (хлебниц/раздвижн), reinforced (усиленн/оцинкован), turnkey (под ключ/установк/фундамент), custom (индивидуальн/на заказ), parnik, polycarbonate, teplica_generic, other
- Гео-разрезы: perm / tyumen / izhevsk / ekb / none — по названиям городов и микрорайонов
- Результаты агрегированы через `sort -k3 -rn` по кликам
